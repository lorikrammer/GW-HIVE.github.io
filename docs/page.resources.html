Tools/Applications

  • GW HIVE - GW instance of the High-performance Integrated Virtual Environment (PMID: 25271953; 26989153)
  • AWS HIVE - AWS instance of the High-performance Integrated Virtual Environment (PMID: 25271953; 26989153)
  • BioCompute Galaxy - Galaxy is an open, web-based platform for data-intensive research.
  • BioCompute Objects Portal (BCOP) - Portal for making BioCompute objects
  • PhyloSNP - generate phylogenetic trees from single-nucleotide variation data (PMID: 24930720)
  • CensuScope - detect taxonomic composition of a metagenomic data set (PMID: 25336203)

Resources

  • BioMuta - browse human disease associated single-nucleotide variations (SNVs) (PMID: 24667251) 
  • BioXpress - is a curated gene expression and disease association database. (PMID: 25819073) 
  • A Reference Viral Database (RVDB) - DB for enhanced virus detection using next-generation sequencing (NGS) technologies
  • Gut Feeling Knowledge Base - Gut feeling knowledgebase is a reference database of healthy human gut microbiome. (DOI: 10.1371/journal.pone.0206484)
  • Filtered NT
  • OncoMX - KB of unified cancer genomics data from integrated mutation, expression, literature, and biomarker databases, accessible through web portal
  • Codon Usage Table (CUT) - DB for accessing codon usage information
  • GlyGen - data integration and dissemination project for carbohydrate and glycoconjugate related data.
